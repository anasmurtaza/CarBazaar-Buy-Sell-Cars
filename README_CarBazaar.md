# 🚗 CarBazaar -- Buy & Sell Cars Website

A responsive car marketplace website built with HTML5, CSS3, Bootstrap 5
and JavaScript.

## Features

-   Responsive UI
-   Hero carousel
-   Car inventory
-   Search & filters
-   Car details modal
-   Sell your car form
-   Contact form

## Technologies

-   HTML5
-   CSS3
-   Bootstrap 5
-   JavaScript
